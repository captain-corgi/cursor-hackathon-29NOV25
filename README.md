# This repo use for cursor hackathon 29 Nov 2025

This is a simple Eletron application that allows you to display your AI Coding Assistant usage.
For now, it only support Claude Code

## How to use

1. Clone this repo
2. Run `npm install` to install the dependencies
3. Run `npm start` to start the application

## How to contribute

1. Fork this repo
2. Create a new branch
3. Commit your changes
4. Push your changes to your fork
